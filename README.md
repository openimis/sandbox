# Sandbox Setup
Sandbox Setup Repository


This repository is a sandbox for testing and prototyping of openIMIS components and tools for interoperability purposes.


The openIMIS Sandbox Setup is a GIZ-led initiative aimed at enhancing the interoperability of the openIMIS platform by creating a secure, scalable testing environment where real-world integration scenarios with digital public goods like openCRVS, DHIS2, and OpenHIE can be demonstrated and validated. This sandbox will support agile development, FHIR-based data exchange, and automated testing to improve alignment with international standards and streamline health financing and social protection workflows. By fostering collaboration across stakeholders and strengthening digital infrastructure, the project will help reduce integration barriers, improve service delivery, and expand equitable access to essential health and social protection services.
